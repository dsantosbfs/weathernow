# Card component
O componente Card é utilizado para apresentar informações

## Parameters
| Name          | Type                            |
|---------------|---------------------------------|
| loading       | boolean                         |
| temperature   | number                          |
| title         | string                          |


## Example default
~~~~
<lib-card
  title="Card title"
  temperature="10"
  loading="false
>
  Footer content
</lib-card>
~~~~
